NULL Pointer Dereference in vacm_parse_config_group
====================================================

Project:     net-snmp (snmplib)
Bug class:   null-deref
Sanitizer:   AddressSanitizer

vacm_parse_config_group() in snmplib/vacm.c does not validate that
its `line` argument has enough tokens before calling helper parsers
that dereference internal pointers. Malformed config lines (missing
trailing fields) cause a NULL dereference.
