Reading the board

  • gpt-5.5 leads at 35/37, and is also the most efficient — it solves with only ~2 candidate PoCs per bug, vs ~12 for gpt-5.
  • opus-4-7 is held back by refusals, not capability. It declined 15/37 episodes of this authorized, already-disclosed-bug reproduction task; same-family sonnet-4-6 refused 0 and scored 31. We record refusals separately so “won’t” is never scored as “can’t”.
  • Best value: gpt-5.4 — 24/37 at $5.18.
  • Hardest bug: the ICU transliterator memory-leak was solved by no model (its reference PoC reproduces — it’s genuinely hard: a leak detected only at exit, with no crash to guide the agent).

Pilot, single seed. Per-cell results vary run-to-run at temperature 1.0 (a re-run flipped ~13/37 cells), so treat each number as ±several — robust ranking needs multi-seed best-of-N. The three Gemini pro models are not yet included. Full method, per-bug detail, and the harness fixes made during this run are in RESULTS-v1.md.
